## About

This is a project for studying library creation and Python.

Basically, it's a tool to help design and simulate multi-phase FIR compensation filters for CIC filters.

It's necessary to parameterize how the CIC filter is designed and then design the FIR filter.

There is a way to automatically generate a Verilog file with the designed filter.
